# Дорожная карта (Roadmap) развития продукта

Данная дорожная карта визуализирует план развития продукта, от первичного прототипа (MVP) до полномасштабной ERP-платформы.

## 🏁 Milestone 1: Стабилизация MVP (Текущий этап)
**Срок:** 1-2 месяца.
**Цели:** Обеспечить стабильную и быструю работу базового функционала CRM.

- [x] **Базовая CRM**: Управление записями (создание, изменениетусов, удаление).
- [x] **Ролевая модель**: Superadmin (все права) и Master (просмотр и работа со своими заявками).
- [x] **Справочники**: Марки, модели авто, каталог услуг.
- [x] **UI/UX улучшения**: Исправление багов прокрутки и свайпов, улучшение отклика интерфейсов.
- [ ] **Оптимизация синхронизации**: Улучшить механизм фоновой синхронизации данных (SyncQueue), добавить оффлайн-режим с поддержкой IndexedDB.
- [ ] **Безопасность**: Базовая валидация токенов и аутентификационных данных на уровне сервера и клиента.

## 🚀 Milestone 2: Аналитика и Финансы
**Срок:** 2-4 месяца.
**Цели:** Предоставить руководству инструменты для анализа эффективности бизнеса.

- [ ] **Дашборды (Расширенные)**: Графики доходов по дням/неделям/месяцам.
- [ ] **Мотивация мастеров**: Автоматический расчет ЗП на основе процента от выполненных услуг.
- [ ] **Касса и отчетность**: Учет доходов и расходов, печать чеков/накладных, интеграция с фискальными регистраторами.
- [ ] **Уведомления**: Push-уведомления и email-оповещения для мастеров о новых заявках.

## 👥 Milestone 3: Клиентский портал и Телефония
**Срок:** 5-7 месяцев.
**Цели:** Автоматизация взаимодействия с клиентами.

- [ ] **SMS и мессенджеры**: Автоматическая отправка сообщений клиентам ("Машина готова", утренняя памятка).
- [ ] **Онлайн-запись (Виджет)**: Виджет на сайт автосервиса для записи клиентов без участия администратора.
- [ ] **Интеграция с Телефонией (SIP)**: Открытие карточки клиента при входящем звонке.
- [ ] **Программа лояльности**: Начисление бонусных баллов клиентам.

## 🎮 Milestone 4: Игротека и Геймификация
**Срок:** Постоянно интегрируется.
**Цели:** Снижение выгорания персонала и удержание пользователей в системе.

- [x] **Интеграция базовых игр**: Flappy Bird, Rider, Snake, BlockBlast, 2048, Tetris, TicTacToe.
- [ ] **Глобальный лидерборд**: Отслеживание рекордов мастеров по станциям.
- [ ] **Система наград**: Выдача "рубинов" и игровой валюты за успешное выполнение заявок в реальности для покупок внутри игр.
- [ ] **Турниры**: Внутренние соревнования между мастерами за виртуальные/реальные призы.

## 🏢 Milestone 5: Сеть автосервисов (Масштабирование)
**Срок:** 8-12 месяцев.
**Цели:** Обеспечение работы многофилиальной структуры.

- [ ] **Склады и инвентаризация**: Учет запчастей, автоматические заявки поставщикам при нехватке на складе.
- [ ] **Филиалы (Локации)**: Разделение данных по точкам (СТО).
- [ ] **Маршрутизация заявок**: Автоматическое распределение нагрузки по мастерам в разных сервисах.
- [ ] **API для интеграции**: Открытый API для 1С и других внешних ERP.
