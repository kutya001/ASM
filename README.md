# Автоматизированная система управления автосервисом (AutoCRM)

Добро пожаловать в репозиторий AutoCRM — современной системы для оптимизации работы автосервисов, управления заявками, мастерами и справочниками. Приложение также включает встроенные игровые механики (игротеку) для геймификации.

## О проекте

Система предоставляет полноценное рабочее пространство (CRM-панель) для администраторов и мастеров, позволяя:
- Создавать и управлять записями на обслуживание.
- Вести базу данных марок и моделей автомобилей, а также список услуг (справочники).
- Отслеживать статусы выполнения работ, оплаты и занятость мастеров.
- Наслаждаться встроенными мини-играми во время перерывов (Тетрис, Змейка, Rider, Flappy Bird и др.).

## Быстрый старт

Проект написан на **Vue 3** с использованием сборщика Vite.

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```

3. Для сборки в продакшен:
   ```bash
   npm run build
   ```
