# Автоматизированная система управления автосервисом (ASM ERP)

**ASM ERP** — современная SPA-система управления автосервисом с поддержкой мультитенантности, оптимизированная для работы на бэкенде **Supabase (PostgreSQL)** с мгновенной синхронизацией данных через **Realtime WebSockets**.

---

## Ключевой функционал 🚀

1. **Мультитенантность (Организации)**:
   - Поддержка создания независимых автосервисов (организаций).
   - **Главный мастер / Владелец** (роль `SenMaster`) регистрирует свой сервис, одобряет заявки обычных мастеров на вступление и полностью управляет штатом и списком услуг.
   - **Мастера** (роль `Master`) регистрируются с возможностью поиска и выбора существующей организации, отправляя заявку на вступление. Они видят только свои записи в рамках своей организации.
2. **Панель записей автосервиса**:
   - Ведение карточек обслуживания клиентов (статусы, автозаполнение цен на услуги, привязка мастеров, марки/модели автомобилей, статус оплаты).
3. **Справочники**:
   - **Глобальные справочники** марок и моделей автомобилей (редактируются исключительно ролью `Superadmin`).
   - **Индивидуальные услуги** и цены для каждого автосервиса (настраиваются владельцем организации).
4. **Аналитика**:
   - Дашборды со статистикой выручки, среднего чека, выполненных работ и активности мастеров.
5. **Игротека**:
   - Встроенные мини-игры для сотрудников во время перерывов (Тетрис, Змейка, 2048, Flappy Bird и др.) с таблицей рекордов.

---

## Стек технологий 🛠️

- **Frontend**: Vue 3 (Composition API / Options API), Pinia (управление состоянием), Tailwind CSS.
- **Backend**: Supabase (PostgreSQL), Supabase Realtime (подписки по WebSocket).
- **Сборщик**: Vite.
- **Деплой**: GitHub Actions + GitHub Pages.

## Быстрый старт локально 💻

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/kutya001/ASM.git
   cd ASM
   ```

2. **Установите зависимости**:
   ```bash
   npm install
   ```

3. **Настройте переменные окружения**:
   Создайте файл `.env` в корне проекта на основе примера `.env.example`:
   ```env
   VITE_SUPABASE_URL="https://your-project.supabase.co"
   VITE_SUPABASE_ANON_KEY="your-anon-key"
   ```

4. **Запустите dev-сервер**:
   ```bash
   npm run dev
   ```
   Приложение откроется по адресу `http://localhost:3000`.

## Автоматический деплой (GitHub Pages) 🌐

В проект интегрирован workflow-файл автодеплоя через GitHub Actions. При каждом пуше в ветку `main` проект автоматически собирается и разворачивается.

**Для активации деплоя**:
1. Перейдите в настройки вашего репозитория на GitHub: **Settings**.
2. В левом меню выберите раздел **Pages**.
3. В блоке **Build and deployment** найдите выпадающий список **Source** и измените значение с *Deploy from a branch* на **GitHub Actions**.
4. После этого сайт станет доступен по адресу `https://kutya001.github.io/ASM/`.
